# PROJECT VANGUARD

## Version
Alpha 0.0.1

## Genre
First-Person Shooter (FPS)

## Vision
Project Vanguard is an original tactical first-person shooter inspired by fast-paced multiplayer action. The game will feature unique maps, original weapons, intelligent AI, and online multiplayer.

## Platforms
- Windows
- Android (planned)
- Web (future)

## Engine
Unity 6 LTS

## Programming Language
C#

## Features

### Player
- Walking
- Sprinting
- Jumping
- Crouching
- Sliding

### Weapons
- Assault Rifle
- Pistol
- Shotgun
- Sniper Rifle
- Grenades

### Game Modes
- Team Deathmatch
- Free For All
- Capture the Flag
- Battle Royale (future)

### Multiplayer
- Matchmaking
- Friends System
- Voice Chat (future)
